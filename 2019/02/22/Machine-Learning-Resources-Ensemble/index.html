Machine Learning Resources Ensemble

2019-02-22
machine learning, book, courses, blog, people

Here in this post, I collected and selected the valuable resources I have encountered related to machine learning.

More recommendation of resources and suggestion of correcting are warmly welcomed.

Hope it helps!

The image source

Papers

General

  • 1950: Computing Machinery and Intelligence by Alain Turing
  • 2015: Nature: Deep Learning by LeCun Y., Bengio Y. and Hinton G.E.

The Breakthrough

  • 1986 Back-propagating: Learning representations by back-propagating errors, Nature
  • 2006 AE: Autoencoder: Reducing the Dimensionality of Data with Neural Networks, Science
  • 2013 VAE: Variational Autoencoder: Auto-Encoding Variational Bayes
  • 2014 GAN: Generative Adversarial Networks

Application

  • MIT - Deep Learning: an application-orientated course
  • 30 Amazing Application of Deep Learning by YARON HADAD
  • Looking Back at Google’s Research Efforts in 2018: 2018 review of Google research

Courses

  • Andrew Ng - Stanford - Machine Learning on Youtube, also available at Coursera with programming assignments.
  • Coursera - Andrew Ng at deeplearing.ai - Deep Learning: course series including:
    • Neural Networks and Deep Learning
    • Improving Deep Neural Networks: Hyperparameter tuning, Regularization and Optimization
    • Structuring Machine Learning Projects
    • Convolutional Neural Networks
    • Sequence Models
  • Coursera - Daphne Koller - Probabilistic Graphical Models
    • Probabilistic Graphical Models 1: Representation
    • Probabilistic Graphical Models 2: Inference
    • Probabilistic Graphical Models 3: Learning
  • Google - Machine Learning Crash Course
  • Mark Schmidt - UBC - Machine Learning, Prof. Schmidt also made a complete course 80 Lectures on Machine Learning
  • Andrew Zisserman - Oxford -Machine Learning: machine learning, optimization, computer vision
  • Eric Xing - CMU - Probabilistic Graphical Models: this course follows up closely the state-of-art progress of related research area.
  • Geoffrey Hinton - U Toronto - Neural Networks for Machine Learning
  • David Duvenaud - U Toronto - Differentiable Inference and Generative Models
  • Roger Grosse - U Toronto - Intro to Neural Networks and Machine Learning
  • David Rosenberg - NYU - Machine Learning and Computational Statistics: Video on Youtube
  • Aston Zhang et al - Amazon - Dive into Deep Learning provided by MxNet Group
  • Pieter Abbeel - Berkeley - Introduction to AI
  • Homemade Machine Learning
  • Sergey Levine - Berkeley - Deep Reinforcement Learning
  • Chris Manning - Stanford - Natural Language Processing with Deep Learning
  • Serge Belongie - Cornell - Applied Machine Learning
  • Coursera - U Toronto - Self-Driving Cars course series including
    • Introduction to Self-Driving Cars
    • State Estimation and Localization for Self-Driving Cars
    • Visual Perception for Self-Driving Cars
    • Motion Planning for Self-Driving Cars
  • Coursera - National Research University Higher School of Economics - Advanced Machine Learning course series including
    • Introduction to Deep Learning
    • How to Win a Data Science Competition: Learn from Top Kagglers
    • Bayesian Methods for Machine Learning
    • Practical Reinforcement Learning
    • Deep Learning in Computer Vision
    • Natural Languague Processing
    • Addressing Large Hadron Collider Challenges by Machine Learning
  • Coursera - U Washington - Machine Learning course series including, learning with practical cases
    • Machine Learning Foundations: A Case Study Approach
    • Machine Learning: Regression
    • Machine Learning: Classification
    • Machine Learning: Clustering & Retrieval
  • Coursera - IBM - Advanced Data Science
    • Fundamentals of Scalable Data Science
    • Advanced Machine Learning and Signal Processing
    • Applied AI with DeepLearning
    • Advanced Data Science Capstone
